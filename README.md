# Job Search Graphing App

This project was built with Visx and G3 graphing libraries and React. It's a fun visualizer of metrics of the job search.

## Getting Started

1. Clone the repo
2. Run `npm install`
3. Run `npm start`

## Built With

- [React](https://reactjs.org/)
- [Visx](https://airbnb.io/visx/)
- [D3](https://d3js.org/)
